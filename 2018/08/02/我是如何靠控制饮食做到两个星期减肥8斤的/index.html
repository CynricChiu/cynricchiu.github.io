长期挂在嘴边的“减肥”计划终于落地，感谢Max的支持！因为自己比较懒，健身什么的全身心的在拒绝，跑步跳绳搞了几次就不了了之了。但是经过多方调查，确认了节食减肥的科学性，并试行之，取得了两个星期减肥8斤的效果，在此做个记录以便以后体重反弹了再做参考…

节食减肥原理

原理很简单，只要个人每天摄入的热量少于他每天消耗的热量，就可以达到减肥的效果。为此首先要计算个人每天消耗的热量。
根据毛德倩提出的基础代谢率计算公式，个人每天消耗的基础总热量为：

Q = ( 48.5 * 体重 + 2954.7 ) / 4.184 * 活动因子   男性
Q = ( 41.9 * 体重 + 2869.1 ) / 4.184 * 活动因子   女性

Q的单位是千卡，俗称大卡，即1大卡=1kcal。其中体重单位是千克，活动因子参考如下

生活方式职业或人群活动因子
休息、坐卧为主生活不能自理的老人、残疾人士1.2
静态生活办公室职员1.41-1.5
坐姿为主、偶尔活动学生、司机、装配工人1.6-1.7
站立、走动为主销售人员、服务员、接待1.8-1.9
重体力为主建筑工人、农民、运动员、运动爱好者2.0-2.4

以我为例，我的体重是65kg，活动因子定为1.6，那么每天的基础消耗约为2335大卡。

节食食谱制定

经过以上计算，意味着每天摄入的食物热量总和不能超过2335大卡。既要管饱，又要兼顾营养，食谱制定是个难题，这里参考豆瓣作者流浪诗人给出的食物热量参照表



食谱一定要定量，所以需要一个厨房电子小秤，用来称量食材的重量。谢谢Max的厨房小秤！爱你！
因为懒的原因，我主要还是选择了诸如黄瓜、胡萝卜、西红柿之类的洗洗就能吃的食材，不过蔬菜切成丁以后真的是满满一碗还没有多少热量，而相比较而言，香蕉不愧是热量之王，一天两根香蕉基本上就不能再吃别的了。。。特别注意的是，像酱油、醋、油之类的烹饪的调料热量也是不容忽视的，所以我做菜的时候基本只是放点盐而不放别的调料。下面贴几张自己的完整的食谱：

节食效果

虽说确实瘦了8斤，但是我开始减肥的目的是为了减肚腩，Max说我减肥减的腿细了，而肚腩只是小了一点。。。不管怎么样，经过验证，科学的节食减肥是可行的，但个人认为不是长久之计，合理安排饮食并结合运动才是往后保持身材的好方法。而像减肚腩这种专门减一个部位的体重的，最好还是通过健身，光靠节食是不管用的，哎。
