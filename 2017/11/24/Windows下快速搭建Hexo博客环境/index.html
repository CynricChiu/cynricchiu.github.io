上传后用浏览器访问 http://用户名.github.io/ 地址即可查看博客，若要将博客绑定至自己的域名，接着如下步骤：
1、修改_config.yml文件里的 url 值为你的域名
2、在source文件夹下新建CNAME文件，内容为你的域名(不带http和www)
3、在github仓库设置页面设置custom domain为你的域名(不带http和www)
修改后再上传一遍博客，等几分钟待DNS生效即可
